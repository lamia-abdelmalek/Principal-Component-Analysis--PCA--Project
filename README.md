# 📊 Principal Component Analysis (PCA) Project

## 📌 Overview

This project demonstrates the implementation and application of **Principal Component Analysis (PCA)** for dimensionality reduction and data visualization.
The notebook walks through the mathematical intuition, implementation steps, and visualization of transformed data.

## 🎯 Objectives

* Understand how PCA works mathematically
* Reduce high-dimensional data into fewer components
* Visualize data in 2D/3D space
* Analyze explained variance

## 🧠 What is PCA?

Principal Component Analysis (PCA) is a dimensionality reduction technique that:

* Transforms data into a new coordinate system
* Maximizes variance along principal components
* Reduces complexity while preserving important information
